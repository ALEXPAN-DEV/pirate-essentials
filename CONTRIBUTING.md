# Contributing

Thank you for your interest in contributing to Pirate Essentials.

## Getting started

1. Fork the repository
2. Clone your fork
3. Create a new branch from `dev`
4. Make your changes
5. Test the installer
6. Open a Pull Request

## Development setup

Clone the repository: git clone https://github.com/ALEXPAN-DEV/pirate-essentials.git
Run the installer: run.bat

## Reporting issues

If you encounter a bug:

1. Open an Issue
2. Describe the problem clearly
3. Include the full console output
4. Attach log files from the `logs/` folder if available

## Pull requests

Before submitting a pull request:

- Ensure the installer runs correctly
- Test your changes in a clean environment
- Follow the existing code style

Thank you for helping improve the project.
